## Quick Start

### Requirements

* `node >= 8`
* `yarn >= 1`

### Clone this repo

```bash
$ git clone https://github.com/epilande/monorepo.git
$ cd monorepo
```

### Install dependencies

```bash
$ yarn install
```

### Launch dev environment

_Navigate to [http://localhost:8080/](http://localhost:8080/)_

```bash
$ yarn start
```
